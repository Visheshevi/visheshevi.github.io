Parallel Processing

Found the determinant of an nxn matrix with the help of parallel processors at the HPC installed in UIC. The determinant is found out by performing the LU decomposition of the matrix. The matrix is divided on a mesh of processors and it is decomposed into two matrices: lower and upper. The final determinant is calculated by adding the two matrices.

Computed the sum of an array of n numbers over p parallel processors. Two different toplogies, ring and hypercube, were used to compute the final sum. At the end, the computation time and communication time are calculated to find the better topology.

Core Design Space Exploration

In this project, I parallelized the fast fourier transform using MPI library. I also changed the configuration of the 'eeg' benchmark to get the Minimum Energy Delay Area Prodduct(EDAP).

I also changed the different parameters of the architecture ranging from the Branch Prediction attributes like the type of predictor, return address stack size and Branch Target Buffer configuration to the number of Functional units like number of integer and Floating point ALUs. By configuring all these parameters I was able to succesfully decrease the overall Energy Delay Product from 7.89 to 5.69 and increase the IPC from 1.523 to 1.6989.

Feature Matching

In this project, I found the features that match between an image and its rotated version using MATLAB. Every function used was made from scratch and no computer vision library was used.

To match the features, first I found out the X and Y derivatives of the image. Then the image was thresholded and rotated. Then used the harris detector across both the thresholded and the rotated thresholded image. Comparing both the results, I got the features that matched in both the image.
